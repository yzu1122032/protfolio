Project Concept

本專案是一款角色扮演的卡牌闖關遊戲，玩家扮演一隻勇敢的小熊，為了救回被外星人綁架的父母，踏上冒險旅程。小熊需要在每一關中擊敗敵人，並從第二關開始可使用隨機抽取的「技能卡牌」來強化自己、突破重重難關，最終成功打敗外星人首領，救出父母。

遊戲故事 : 在一片寧靜的森林深處，小熊和他的父母住在一間溫馨的小木屋裡，過著平凡而幸福的生活。每天小熊都會在森林裡玩耍，然而這份平靜的日子在某天被突如其來的事件打破—— 那天，小熊像往常一樣外出採蜂蜜，卻突然看到一束奇異的藍光從家的方向閃現。他急忙趕回去卻發現一艘神秘的UFO正懸停在小木屋上空，而他的父母已被外星人抓走！小熊奮力衝向家門，但只能眼睜睜看著UFO載著父母消失在天際⋯⋯

遊戲目的 : 透過遊戲讓玩家體驗愛與勇氣的重要，培養問題解決能力與持續挑戰的精神，並使用簡單易懂的卡牌玩法，讓玩家不需要複雜的操作就能享受策略樂趣。

Card Function Design

卡牌設計以功能分類為核心，包含攻擊、防禦、強化與其他特殊能力等類型， 透過不同效果讓玩家在遊戲中能夠策略性地選擇與搭配。

在視覺上使用顏色與圖示區分卡牌種類，讓玩家能快速辨識功能， 同時維持整體風格一致性。

普通卡 : 增加攻擊力10點，不能疊加，沒有冷卻時間。

攻擊卡 : 增加15點的攻擊力，最多可疊加2次，沒有冷卻時間。

強化卡 : 增加自身50點的攻擊力和防禦力，可持續2回合，有3回合的冷卻時間。

隱形卡 : 使用後可無視一回合的對手攻擊，有3回合的冷卻時間。

回血卡 : 增加25點的生命值，有3回合的冷卻時間。

防禦卡 : 增加25點的防禦力，只生效一回合，沒有冷卻時間。
